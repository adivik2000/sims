Getting Started Setting up a District

Step Manual CSV
1. Login as a district admin.
2. Edit District Settings.
3. Add schools. schools.csv
4. Add Users. users.csv, regular_users.csv
5. Designate School Admins school_admins.csv, (schools must be assigned manually)
6. Designate Content Admins. content_admins.csv
7. Add Students. students.csv
8. Add Enrollments. enrollments.csv
9. Choose Users with All Students in District. all_students_in_district.csv
The users in step 9 can now use SIMS.
10. Add User School Assignments. user_school_assignments, staff_assigments.csv (optional)
11. Choose users with All Students in a School. all_students_in_school.csv
12. Add Groups. groups.csv
13. Assign Students to Groups. student_groups.csv
14. Assign Users to Groups. user_groups.csv
15. Add System Flags. system_flags.csv
16. Create Extended Profiles. ext_arbitraries.csv, ext_siblings.csv, ext_summaries.csv, and ext_test_scores.csv
